## u-modal 模态框
使用 Vue.js 框架（是基于 uni-app 或 uView UI 的组件）实现的一个模态框（Modal）组件。
* * *
### 一、代码解析

    <u-modal 
        v-model="showCancelModal" 
        title="取消编辑" 
        content="确定要取消编辑并重新拍照吗？"
        show-cancel-button
        @confirm="handleCancel"
        @cancel="showCancelModal = false">
    </u-modal>
    
#### 1\. `v-model="showCancelModal"`

*   这是一个双向绑定，用于控制模态框的显示与隐藏。
*   当 `showCancelModal` 为 `true` 时，模态框显示；为 `false` 时隐藏。

#### 2\. `title="取消编辑"`

*   设置模态框的标题为“取消编辑”。

#### 3\. `content="确定要取消编辑并重新拍照吗？"`

*   设置模态框的内容文本，提示用户是否确认取消编辑并重新拍照。

#### 4\. `show-cancel-button`

*   表示在模态框中显示“取消”按钮。

#### 5\. `@confirm="handleCancel"`

*   当用户点击“确认”按钮时，会触发 `handleCancel` 方法。

#### 6\. `@cancel="showCancelModal = false"`

*   当用户点击“取消”按钮时，将 `showCancelModal` 设置为 `false`，从而关闭模态框。

* * *

### 四、完整示例（Vue + uView UI）

    <template>
      <view>
        <button @click="showCancelModal = true">取消编辑</button>
    
        <u-modal 
            v-model="showCancelModal" 
            title="取消编辑" 
            content="确定要取消编辑并重新拍照吗？"
            show-cancel-button
            :confirm-button-style="{ backgroundColor: '#ff4949', color: '#fff' }"
            @confirm="handleCancel"
            @cancel="showCancelModal = false">
        </u-modal>
      </view>
    </template>
    
    <script>
    export default {
      data() {
        return {
          showCancelModal: false
        };
      },
      methods: {
        handleCancel() {
          // 在这里执行取消编辑并重新拍照的逻辑
          console.log("用户确认取消编辑并重新拍照");
          this.showCancelModal = false;
        }
      }
    };
    </script>
