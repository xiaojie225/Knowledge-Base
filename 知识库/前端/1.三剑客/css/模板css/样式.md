## 文本处理
### 多行文本溢出隐藏

.article-title {
  display: -webkit-box;           /* 使用弹性盒子布局 */
  -webkit-box-orient: vertical;   /* 子元素垂直排列 */
  -webkit-line-clamp: 2;          /* 限制显示行数为 2 行 */
  overflow: hidden;               /* 隐藏溢出内容 */
  text-overflow: ellipsis;        /* 溢出部分显示省略号 */
}

white-space: nowrap;
无论容器有多窄，都不要将元素内的文本折行。

### 英文换行优化：
style="max-width: 80%;word-break: break-word;text-align: center"

word-break: break-word防止长英文单词溢出

max-width: 80%限制文本容器宽度

使用 &nbsp; 硬空格
```
 <text >
  {{$t('common.please')}}&nbsp;
  <span style="color: #D7F548;">{{$t('common.click')}} </span>&nbsp;
  {{$t('common.information')}}
</text>
```

## 图片处理
img {
width: 100%; /* 设置图片宽度为容器的100% */
height: auto; /* 设置图片高度自动，以保持图片比例 */
object-fit: contain; /* 保持原有尺寸比例，内容被缩放 */
}
