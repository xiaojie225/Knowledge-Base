# 前端三剑客

HTML、CSS、JavaScript 是前端开发的三大基石，这里记录学习这三大技术的笔记。

## 📖 学习内容

### 🏗️ HTML - 网页结构

HTML（HyperText Markup Language）是网页的骨架，定义了网页的结构和内容。

**主要学习内容：**
- HTML 基础语法
- 语义化标签
- 表单与表格
- HTML5 新特性

### 🎨 CSS - 网页样式

CSS（Cascading Style Sheets）负责网页的样式和布局，让网页变得美观。

**主要学习内容：**
- CSS 选择器
- 盒模型
- Flexbox 布局
- Grid 布局
- 响应式设计

### ⚡ JavaScript - 网页交互

JavaScript 是网页的灵魂，实现网页的动态交互效果。

**主要学习内容：**
- JavaScript 基础语法
- DOM 操作
- 事件处理
- ES6+ 新特性
- 异步编程

## 🎯 学习路线

1. **基础阶段**：掌握 HTML/CSS 基础，能够搭建静态页面
2. **进阶阶段**：深入学习 CSS 布局，掌握响应式设计
3. **高级阶段**：精通 JavaScript，能够开发复杂交互

## 💡 实践项目

- 个人简历页面
- 响应式博客布局
- 交互式表单验证
- 待办事项应用
