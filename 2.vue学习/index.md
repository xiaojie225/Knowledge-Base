# Vue 学习笔记

Vue.js 是一套用于构建用户界面的渐进式 JavaScript 框架。这里记录学习 Vue 2 和 Vue 3 的笔记。

## 🚀 Vue 简介

Vue.js 是目前最流行的前端框架之一，以其轻量、灵活、易学的特点深受开发者喜爱。

**核心特点：**
- 响应式数据绑定
- 组件化开发
- 虚拟 DOM
- 丰富的生态系统

## 📚 学习内容

### Vue 基础

- Vue 实例与生命周期
- 模板语法与指令
- 计算属性与侦听器
- Class 与 Style 绑定

### 组件开发

- 组件基础与通信
- 插槽（Slots）
- 动态组件与异步组件
- 组件库开发

### Vue Router

- 路由配置与导航
- 路由守卫
- 动态路由
- 路由懒加载

### 状态管理

- **Vuex**（Vue 2 主流）
  - State、Getters、Mutations、Actions
  - 模块化
  
- **Pinia**（Vue 3 推荐）
  - 更简洁的 API
  - TypeScript 支持
  - Devtools 支持

### Vue 3 新特性

- Composition API
- `<script setup>` 语法糖
- Teleport 组件
- Suspense 组件
- 更好的 TypeScript 支持

## 🎯 学习路线

1. **入门阶段**：掌握 Vue 基础语法和概念
2. **进阶阶段**：深入组件开发和路由管理
3. **实战阶段**：开发完整的 Vue 项目
4. **高级阶段**：学习 Vue 3 和 Composition API
