# 数组

数组是最基础的数据结构之一，在内存中连续存储相同类型的元素。

## 数组的特点

- **连续存储**：元素在内存中连续存放
- **随机访问**：可以通过索引快速访问任意元素
- **固定大小**：大多数语言中数组大小固定（动态数组除外）

## 时间复杂度

| 操作 | 时间复杂度 |
|------|----------|
| 访问元素 | O(1) |
| 搜索元素 | O(n) |
| 插入元素 | O(n) |
| 删除元素 | O(n) |

## JavaScript 数组示例

```javascript
// 创建数组
const arr = [1, 2, 3, 4, 5];

// 访问元素
console.log(arr[0]); // 1

// 添加元素
arr.push(6); // [1, 2, 3, 4, 5, 6]

// 删除元素
arr.pop(); // [1, 2, 3, 4, 5]
```

## 常见应用

- 存储有序数据
- 实现其他数据结构（栈、队列等）
- 动态规划问题
